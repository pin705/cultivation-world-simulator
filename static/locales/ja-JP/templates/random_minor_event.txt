あなたは仙道世界のサンドボックス・シミュレーションに参加しています。ある人物に対して、短い「ランダム小事件」を生成してください。

キャラクター情報: {avatar_info}
現在地: {location}
世界状態: {world_info}
事件の原因: {cause}
対象人物 / 補足: {target_info}

要件:
1. 上記情報と仙道世界の背景をもとに、20〜50 字程度のごく短い事件描写を作ること。面白くても、地味でも、単なる日常でもよい。
2. 対象人物が存在し、人物情報内に過去の出来事や関係が含まれるなら、その背景を自然に小事件へ織り込むこと。
3. 小事件そのものだけを描写し、強制的な数値変動や大きな結果（例: 「重傷を負った」「神器を得た」など）は書かないこと。
4. 対象人物が「名無しの通りすがり」である場合は、仙道世界にふさわしい簡単な背景を自然な範囲で補うこと。
5. 応答は必ず以下の厳密な JSON 形式にすること:
```json
{{
  "event_text": "ここに事件文を入れる"
}}
```
